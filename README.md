# BEclear-CL

## Description
Correction of batch effects with [BEclear](https://bioconductor.org/packages/release/bioc/html/BEclear.html) as a command line tool.

## Installation

```bash
chmod +x BEclearCL.R
```

## Usage example

Accessing the help pages:

```bash
./BEclearCL.R -h
```

```bash
cat test.txt | ./BEclearCL.R -w
```
